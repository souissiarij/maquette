🌐 Maquette Web - Gestion de la Vie Associative | ENICarthage
Ce projet est une maquette responsive d’un site web destiné à gérer et promouvoir la vie associative de l’ENICarthage. Il a été conçu en utilisant HTML, CSS, JavaScript et le framework Bootstrap, avec une attention particulière portée à l’ergonomie et à l’expérience utilisateur.

🛠️ Technologies utilisées
HTML5

CSS3

JavaScript

Bootstrap 5

(optionnel : Font Awesome, Google Fonts)

✨ Fonctionnalités de la maquette
Page d’accueil dynamique présentant les clubs et événements à venir

Fiches descriptives pour chaque club (objectifs, membres, projets, contact)

Formulaire d’enrôlement dans un club

Section d’administration (maquette) pour les responsables de clubs

Interface responsive compatible mobile, tablette et desktop

📌 Objectif du projet
Ce projet vise à simuler l'interface d'un futur site web de gestion de la vie associative. Il sert de base pour le développement frontend complet et peut être connecté ultérieurement à un backend pour gérer les données en temps réel.
